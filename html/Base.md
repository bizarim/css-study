html 기본 문서 구조
``` html
<!doctype html> : 현재 문서가 HTML5 언오로 작성된 웹 문서라는 뜻입니다.

<html>
웹 문서의 시작과 끝을 나타내는 태그
</html>
<head>
웹 브로아저가 웹 문서를 해석하기 우ㅐ해 필요한 정보들을 입력하는 부분
실제 문서 내용이 아니기 때문에 문서 제목만 브라우저 창에 표시되고,
나머지는 웹 브라우저 화면에 표시 되지 않는다.
스타일 및 스크립트 등이 포함 되기도 한다.
</head>
<body>
실제로 웹 브라우저 화면에 나타날 내용입니다.
앞으로 우리가 공부하게 될 HTML 태그들은 대부분 <body>태그 안에 들어간다.
</body>
```
